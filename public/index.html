Permissions
Who can start a /watch or /movienight room (one toggle covers both — a role that can vote on a movie can also start the room that plays it). Guests can never start rooms. Per-user overrides on Superadmin → Users (the Watch Party column).
Sync strategy
When you start a Watch Together room, the server tries to resolve the URL into a directly-playable stream so all viewers stay in sync. Different sites need different approaches — switch this if a source isn't working. Auto tries them in order. Iframe-only skips resolution and uses a click-to-start embed (no sync, but lowest-latency for the host).
Capture proxy + CDP timing
The capture proxy and CDP sniffer share these knobs. Higher capture resolution / framerate is heavier on CT 109's CPU; the default (720p / 30 fps) is the safe choice. CDP timeout = how long the sniffer waits for a page's first stream URL before giving up.
